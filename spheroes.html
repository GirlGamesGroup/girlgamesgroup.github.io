SP Heroes

SP Heroes is a single player action game for Windows platform and FIESP’s building where the player can help to save a building in São Paulo by putting out the fire.

  • Platform: FIESP Building/PC
  • Genre: Action
  • Engine: Unity
  • Mode: Singleplayer
  • Programmer: Perla Peralta, Clara Cerqueira, Juliana Montes
  • Author: Perla Peralta, Marina Peripato, Isabella Rodolfo, Vania Alvis, Clara Cerqueira
  • Game Artist: Clara Cerqueira, Perla Peralta, Isabella Rodolfo, Marina Peripato, Vania Alvis
  • Game Designer: Isabella Rodolfo, Marina Peripato, Clara cerqueira, Perla Peralta, Vania Alvis
